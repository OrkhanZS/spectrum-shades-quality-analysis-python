## Project Overview
This project analyzes factors contributing to inconsistent concrete color quality at Spectrum Shades LLC.

## Business Problem
The company has received increasing customer complaints regarding inconsistent color quality in concrete pigment products.

## Objectives
- Clean and validate production data
- Analyze supplier impact on product quality
- Perform statistical analysis
- Measure correlation between pigment quantity and product quality

## Methods Used
- Data Cleaning & Validation
- Grouped Aggregation
- Filtering & Conditional Analysis
- Mean & Standard Deviation
- Pearson Correlation

## Tools
- Python
- Pandas
- Statistical Analysis

## Outcome
The analysis identifies potential factors influencing product quality variations and provides data-driven insights to improve production reliability.
